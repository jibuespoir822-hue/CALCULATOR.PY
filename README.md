# 🧮 Python Calculator (Tkinter)

## 📌 Description
This is a fully functional calculator built using Python and Tkinter.  
It supports basic operations and includes a user-friendly graphical interface.

## 🚀 Features
- Addition, Subtraction, Multiplication, Division
- Percentage calculation
- Input validation
- Clean and simple UI

## 🖥️ Technologies Used
- Python
- Tkinter

## ▶️ How to Run
1. Make sure Python is installed
2. Run the file:
   python calculator.py
